02

Install app

brew install --cask jarvis-app

03

Choose roles

jarvis service install brain worker

04

Pair devices

jarvis pair kitchen-pi --json

Public-ready

Secrets stay local. Devices ask to join.

Per-device tokens are bound to device ids.

Browser profiles, OAuth caches, user files, and worker jobs are ignored.

Repos only go public after the release scan passes.

Updates

Brew owns the runtime. The app owns the experience.

brew update brew upgrade jarvis brew upgrade --cask jarvis-app jarvis service restart brain
